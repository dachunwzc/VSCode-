变异方法 ------>>>> 会改变被调用的原始数组

push() 方法可向数组的末尾添加一个或多个元素，并返回新的长度。

  1.         let arrayPush = []
  2.         arrayPush.push(1)
  3.         arrayPush.push(3)
  4.         arrayPush.push(2)
  5.         console.log(arrayPush) //=> [1,3,2]

pop() 方法用于删除并返回数组的最后一个元素。

  1.         let arrayPop = [1, 2, 3, 6]
  2.         arrayPop.pop()
  3.         let arrayPopb = arrayShift.pop()
  4.         console.log(arrayPopb) //=> 6
  5.         console.log(arrayPop) //=> [1, 2, 3]

shift() 方法用于把数组的第一个元素从其中删除，并返回第一个元素的值。

  1.         let arrayShift = [1, 2, 3, 6]
  2.         arrayShift.shift()
  3.         let b = arrayShift.shift()
  4.         console.log(b) //=> 2
  5.         console.log(arrayShift) //=> [2, 3, 6]

unshift() 方法可向数组的开头添加一个或更多元素，并返回新的长度。

  1.         let arrayUnshift = [3, 6]
  2.         arrayUnshift.unshift(1)
  3.         console.log(arrayUnshift) //=> [1,3,6]

sort() 方法用于对数组的元素进行排序。

  1.         // 第一个参数是插入位置，第二个参数是删除若干个元素，设置0为不删除任何元素，第三个及以后是插入的值

reverse() 方法用于颠倒数组中元素的顺序

  1.         let arrayReverse = [1, 9, 4, 2]
  2.         arrayReverse.reverse()
  3.         console.log(arrayReverse) //=> [2, 4, 9, 1]

替换数组 ——>>>> 不会改变被调用的原始数组

filter() 方法创建一个新的数组，新数组中的元素是通过检查指定数组中符合条件的所有元素。

  1.         let arrayFilter = [1, 9, 4, 2]
  2.         let arrayFilter2 = arrayFilter.filter(function (arrayFilter) {
  3.           return arrayFilter >= 2;
  4.         })
  5.         console.log(arrayFilter2); //[9, 4, 2]

concat() 方法用于连接两个或多个数组。

  1.         let arrayConcat = [1, 9, 4, 2]
  2.         let arrayConcat2 = [6, 7, 8, 3]
  3.         let arrayConcat3 = arrayConcat.concat(arrayConcat2)
  4.         arrayConcat3.sort(function (a, b) {
  5.           return a - b;
  6.         })
  7.         console.log(arrayConcat3); //[1, 2, 3, 4, 6, 7, 8, 9]

slice() 方法可从已有的数组中返回选定的元素。

  1.         //slice()接受一个或两个参数，即要返回项的起始和结束位置。当只给slice()传递一个参数时，该方法返回从该参数指定位置开始到当前数组末尾的所有项。
  2.         let arraySlice = [1, 2, 3, 4, 6, 7, 8, 9]
  3.         let arraySlice1 = arraySlice.slice(4)
  4.         let arraySlice2 = arraySlice.slice(3,6)
  5.         console.log(arraySlice); //[1, 2, 3, 4, 6, 7, 8, 9]
  6.         console.log(arraySlice1); //[6, 7, 8, 9]
  7.         console.log(arraySlice2); //[4, 6, 7]

其他

split() 方法用于把一个字符串分割成字符串数组。

  1.         let stringNum = '1,2,3,4,5,6,7,8'
  2.         let arrayNum = stringNum.split(',')
  3.         console.log(arrayNum); // ["1", "2", "3", "4", "5", "6", "7", "8"]

Mac环境下npm run serve 提示 node_modules/.bin/vue-cli-service: Permission denied问题解决方案

之前windows打开的项目，用mac打开的时候报错：网上找了解决方案，为了方便查询，再记录一下子

 

: /Users/dongjingya/Documents/workspace/vue-manage-system/node_modules/.bin/vue-cli-service: Permission denied

npm ERR! code ELIFECYCLE

npm ERR! errno 126

npm ERR! vue-manage-system@4.2.0 serve: `vue-cli-service serve`

npm ERR! Exit status 126

npm ERR! 

npm ERR! Failed at the vue-manage-system@4.2.0 serve script.

npm ERR! This is probably not a problem with npm. There is likely additional logging output above.

 

解决方案，亲自试了，好用

打开mac终端，输入chmod 777 /Users/此处是项目根目录/node_modules/.bin/vue-cli-service 执行即可。

 

参考原文 https ://blog.csdn.net/yangzhongwei1031/article/details/104488929/

swift：在字符串中使用％s（格式：...）

[英]swift : use of %s in String(format: …)

本文翻译自  Julien  查看原文  2017-07-12  4272    string/ format/ swift           

I would like to format a string with another string like this:

我想用另一個字符串格式化一個字符串,如下所示:

var str = "Hello, playground"
print (String(format: "greetings %s", str))

This leads to this beautiful result:

這導致了這個美麗的結果:

greetings 哰૧

I tried with %@ and it works but, as I'm getting format string from another programming language, I would like, if possible to use the %s tag. Is there a way to ?

我嘗試使用%@並且它有效但是,因為我從另一種編程語言獲得格式字符串,我想,如果可能的話,使用%s標簽。有辦法嗎?

2 个解决方案

#1

5  

Ok, I will replace %s by %@ . Python is my friend.

好的,我會用%@替換%s。 Python是我的朋友。

Thanks!

#2

0  

If the format is from an external source, you can convert it to replace occurences of %s with %@:

如果格式來自外部源,您可以將其轉換為使用%@替換%s的出現:

So, instead of:

所以,而不是:

String(format: "greetings %s", str)

You do:

String(format: "greetings %s".replacingOccurrences(of: "%s", with: "%@"), str)
